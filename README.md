# StandSitReminder

Stand/Sit Reminder is a productivity-focused Chrome extension designed to promote better ergonomic habits for desk workers. This extension helps users maintain a healthier work routine by sending timely reminders to alternate between sitting and standing positions throughout the workday.

## Features
- Customizable timer intervals for standing and sitting
- Non-intrusive desktop notifications
- Simple and clean user interface
- Productivity tracking statistics
- Ergonomic health tips and guidance

## Installation
The extension can be installed directly from the Chrome Web Store.

## Usage
After installation, set your preferred sitting and standing intervals. The extension will automatically remind you when it's time to change positions, helping you maintain better posture and reduce the health risks associated with prolonged sitting.
